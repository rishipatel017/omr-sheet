# 🚀 Why OMR Pro?

OMR Pro is a lightweight, browser-based Optical Mark Recognition (OMR) Sheet Generator and Evaluation System developed using **AngularJS 1.8**.

Unlike traditional OMR software, this project works entirely inside the browser without requiring:

- ❌ Backend Server
- ❌ Database
- ❌ User Authentication
- ❌ Installation
- ❌ Internet Connection (after loading)

Everything happens locally, making the application extremely fast, portable, and privacy-friendly.

# ⚙️ How It Works

## Step 1

Configure your OMR sheet.

Choose

- Number of Questions
- Number of Options
- Number of Columns

Example

```text
Questions : 100

Options : 4

Columns : 2
```

Click

```text
Generate
```

---

## Step 2

The application instantly creates a professional OMR sheet.

Example

```text
001   ● A   ○ B   ○ C   ○ D

002   ○ A   ● B   ○ C   ○ D

003   ○ A   ○ B   ● C   ○ D

004   ○ A   ○ B   ○ C   ● D
```

Students simply fill the bubbles.

---

## Step 3

Paste the official answer key.

Supported formats

```text
A,B,C,D,A,C,B,D
```

or

```text
A B C D A C B D
```

No formatting restrictions.

---

## Step 4

Press

```text
Evaluate
```

OMR Pro automatically calculates

- Correct Answers
- Wrong Answers
- Blank Questions
- Final Score
- Percentage
- Grade
- Pass / Fail Status

within milliseconds.

# ⚡ Evaluation Formula

Default marking scheme

| Action | Marks |
|---------|-------|
| ✅ Correct | +4 |
| ❌ Wrong | -1 |
| ⭕ Blank | 0 |

The marking scheme is fully customizable according to your examination requirements.
